## Overview

This portfolio website serves as my digital resume and project showcase. It reflects my passion for web development, clean design, and continuous learning. The site features a sliding door intro animation, a moving profile picture with GIF background, and a fully responsive layout that works on all devices.

## Features

- **Sliding Door Intro Animation** – A cinematic entrance effect that reveals the main content
- **Animated Profile Picture** – Floating animation with a circular gradient frame
- **GIF Background** – Subtle animated background with blur effect
- **Fully Responsive Design** – Optimized for desktop, tablet, and mobile devices
- **Smooth Scrolling Navigation** – Easy navigation between sections
- **Project Showcase** – Display your work with screenshot previews
- **Contact Section** – Social media links with hover animations
- **Dark/Green Aesthetic** – Custom color scheme with modern glassmorphism effects
